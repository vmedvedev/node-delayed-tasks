## About
Delayed tasks implementation with Node.js and Redis.

#### Build docker image:
```bash
$ docker-compose build
```

#### Run server:
```bash
$ docker-compose up
```

#### Run tasks by http request:
```bash
$ curl http://localhost:8080?message=Hello+World
```
